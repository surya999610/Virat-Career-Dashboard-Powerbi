# 🏏 Virat Career Dashboard (Power BI)

## 📌 Brief Summary  
An interactive Power BI dashboard that analyzes the complete cricket career performance of Virat Kohli using data-driven insights and visual storytelling.

---

## 📖 Overview  
This project presents a comprehensive analysis of Virat Kohli’s cricket career across different formats (ODI, Test, T20). The dashboard is designed to provide meaningful insights into his performance trends, achievements, and consistency over the years.

Using Power BI, raw cricket data has been transformed into an interactive and visually appealing dashboard that allows users to explore player statistics dynamically.

## 🛠️ Tools & Technologies  
- **Power BI** – Dashboard creation & visualization  
- **Microsoft Excel / CSV** – Data cleaning and preprocessing  
- **DAX (Data Analysis Expressions)** – Calculations & measures  
- **Power Query** – Data transformation  

---

## 🔍 Key Insights  
- Virat Kohli shows strong consistency across all formats with high averages  
- Peak performance years can be identified through year-wise run trends  
- Centuries and half-centuries indicate sustained excellence over time  
- Performance varies based on opponent and venue conditions  
- Boundary analysis (4s & 6s) highlights scoring patterns  

---

## 📊 Dashboard  
The dashboard includes:  
- KPI cards (Total Runs, Average, Strike Rate, Centuries)  
- Year-wise performance trends  
- Format-wise comparison (ODI, Test, T20)  
- Opponent-wise analysis  
- Interactive filters (slicers) for deep exploration
